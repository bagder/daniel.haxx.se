Building a PVR with Linux on EPIA M10000

Goal: Build a silent and not too ugly piece of equipment that can fit and be used in the living-room for watching movies (downloaded and DVD) and TV and listen to music. It should also be possible to record TV to disk, on demand and using a timer. Maneuverable using a remote.

Hardware used for this so far:

TV-tuner card: Hauppauge WinTV Nicam Stereo PCI card because it has a chip set supported by Linux, it is cheap and has a remote control that seems to be supported by Linux.

Ordered a TV-tuner card from Dustin on 2003-07-31 (SEK 595). I'll start experimenting with that on a plain Linux computer first.

Motherboard and CPU

VIA EPIA M10000 1GHZ C3, Mini-ITX.

Case

(Casetronic 2699R), fanless and neat. External power. PCI riser card included. close-up on the front
